HOW TO USE IT:

1) Upload the ‘anzidesign-icons.css’ file and ‘fonts’ folder to your server

2) Add the CSS file into the <head> section of your site:
<link href="http://yoursite.com/anzidesign-icons.css" rel="stylesheet">

3) Add the icon markup in your page:
<span class="ti-download"></span>

All icons can be viewed at: http://www.anzidesign.web.id
